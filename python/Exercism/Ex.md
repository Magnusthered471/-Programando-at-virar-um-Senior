# Estudando com Exercism

Este repositório reúne meus exercícios e desafios resolvidos no **Exercism.io**.  
Cada arquivo representa uma prática concluída em diferentes linguagens, como **Python, JavaScript, Bash, C++, Java, HTML e CSS**.

## 🎯 Objetivo
- Usar o Exercism como plataforma de **desafios práticos**.  
- Testar meu intelecto e aprendizado com problemas reais.  
- Construir um portfólio público que mostre disciplina e consistência.  
- Fazer meu **GitHub/Gist "sangrar verde"** com commits diários, provando dedicação.

## 📂 Estrutura
- Pastas organizadas por linguagem (`python/`, `javascript/`, `bash/`, `cpp/`, etc.).  
- Cada arquivo corresponde a um desafio resolvido.  
- A evolução vai do **nível iniciante ao avançado**, mostrando progresso contínuo.

## 🚀 Motivação
Não estou apenas estudando teoria: estou **botando a mão na massa** todos os dias.  
O Exercism é meu campo de treino, onde cada solução é um passo rumo ao domínio das linguagens e à construção de um portfólio sólido.
