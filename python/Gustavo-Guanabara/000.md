# 1. Seja um Programador

* **Mentalidade:** A programação é uma competência essencial para a humanidade, com alta demanda futura.
* **Mitos:** Não é necessário ser um mestre em matemática ou inglês fluente. O principal requisito é a **prática constante**.
* **Obstáculos:** O maior inimigo é a falta de compromisso e a pressa de copiar código em vez de aprender a criar.

## 2. Para que serve o Python?

* **História:** Linguagem de alto nível, interpretada e extremamente legível.
* **Uso:** Versatilidade para web, desktop, automação, IA e análise de dados.
* **Pronúncia:** Deve ser pronunciada seguindo a fonética em inglês (*páithon*).

## 3. Instalando o Python3 e o IDLE

* **Ambiente:** Configuração inicial no site oficial (`python.org`).
* **IDLE:** Utilização do interpretador padrão, que é ideal para primeiros testes e entendimento básico.

## 4. Primeiros comandos em Python3

* **Prática:** Primeiro contato com o comando `print()`.
* **Sintaxe:** Introdução à manipulação de strings e regras básicas de escrita de código para evitar erros de sintaxe.

## 5. Instalando o PyCharm e o QPython3

* **Ferramentas:** Transição do IDLE para o **PyCharm** (IDE profissional), que oferece mais recursos para projetos complexos.
* **Mobilidade:** Introdução ao **QPython3** para executar scripts Python diretamente em dispositivos Android.

## 6. Tipos Primitivos e Saída de Dados

* **Objetivo:** Compreender os quatro tipos primitivos de dados (int, float, bool e str) e realizar a interação básica entre o usuário e o programa.

* **Entrada de Dados:** Utilização da função input() para capturar informações fornecidas pelo usuário, que por padrão são sempre lidas pelo Python como o tipo str (texto).

* **Conversão de Tipos:** Necessidade de utilizar funções de conversão, como int() para números inteiros ou float() para números reais, caso seja necessário realizar cálculos matemáticos com os dados recebidos.

* **Saída de Dados:** Uso da função print() combinada com técnicas de formatação (como f-strings ou o método .format()) para exibir os dados processados de maneira organizada e legível na tela.
  
## 7. Operadores Aritméticos

* **Objetivo:** Aprender a manipular números e realizar cálculos matemáticos dentro do Python.  

* **Operadores Principais:** Introdução aos operadores básicos como + (soma), - (subtração), * (multiplicação), / (divisão), ** (potência), // (divisão inteira) e % (resto da divisão).  

* **Ordem de Precedência:** Compreensão da hierarquia de execução matemática, que define a ordem correta das operações (parênteses, potência, multiplicação/divisão, soma/subtração).  

* **Aplicação:** Utilização desses operadores para processar dados numéricos e obter resultados calculados em programas.
