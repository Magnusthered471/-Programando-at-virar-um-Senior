# 1. Seja um Programador

* **Mentalidade:** A programação é uma competência essencial para a humanidade, com alta demanda futura.
* **Mitos:** Não é necessário ser um mestre em matemática ou inglês fluente. O principal requisito é a **prática constante**.
* **Obstáculos:** O maior inimigo é a falta de compromisso e a pressa de copiar código em vez de aprender a criar.

## 2. Para que serve o Python?

* **História:** Linguagem de alto nível, interpretada e extremamente legível.
* **Uso:** Versatilidade para web, desktop, automação, IA e análise de dados.
* **Pronúncia:** Deve ser pronunciada seguindo a fonética em inglês (*páithon*).

## 3. Instalando o Python3 e o IDLE

* **Ambiente:** Configuração inicial no site oficial (`python.org`).
* **IDLE:** Utilização do interpretador padrão, que é ideal para primeiros testes e entendimento básico.

## 4. Primeiros comandos em Python3

* **Prática:** Primeiro contato com o comando `print()`.
* **Sintaxe:** Introdução à manipulação de strings e regras básicas de escrita de código para evitar erros de sintaxe.

## 5. Instalando o PyCharm e o QPython3

* **Ferramentas:** Transição do IDLE para o **PyCharm** (IDE profissional), que oferece mais recursos para projetos complexos.
* **Mobilidade:** Introdução ao **QPython3** para executar scripts Python diretamente em dispositivos Android.
