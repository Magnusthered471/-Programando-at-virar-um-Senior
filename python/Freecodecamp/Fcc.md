# Estudando Python com FreeCodeCamp

Este repositório reúne minhas práticas e exercícios realizados no **FreeCodeCamp**.  
Cada arquivo (`fcc1.py`, `fcc2.py`, ...) representa uma lição concluída e serve como registro da minha evolução.

## 🎯 Objetivo
- Usar o FreeCodeCamp como meu **bootcamp gratuito**.  
- Testar meu intelecto e aprendizado com código simples e direto.  
- Mostrar minha dedicação em preencher ao menos um terço do meu portfólio com trabalho consistente.  
- Fazer meu **GitHub/Gist "sangrar verde"** com commits diários, provando disciplina e presença constante.

## 📂 Estrutura
- Arquivos numerados (`fcc1.py`, `fcc2.py`, ...) → lições e desafios de Python.  
- Novas pastas e arquivos virão conforme avanço em **JavaScript, HTML, CSS, Bash, C++** e outras linguagens.  

## 🚀 Motivação
Não estou apenas estudando teoria: estou **praticando diariamente** e registrando cada passo.  
Esse repositório é a prova da minha **dedicação e esforço contínuo**, sem descanso, para evoluir como programador e construir um portfólio sólido.
